It is common to use the on Hover CSS feature to do something to a screen elements attributes. You need to learn how to change a style attribute directly using JavaScript.
CSS features a long list of pseudo-classes that can be used to change the CSS of an element for a given state, like when something has been clicked, the mouse is hovering over it, or an item is enabled/disabled. As seen in previous topics, these properties can also be changed using JavaScript (or more commonly jQuery). 
button:hover {
    color: red;
}

a:visited {
    color: orange;
}

input:focus::-webkit-input-placeholder {
    color: green;
    text-align: center;
}

These CSS changes are only possible when trying to alter the current element being interacted with or its children. To influence another element entirely, we use JavaScript. Every attribute set in CSS can be modified this way. 

var elmVis = document.getElementById("toggleVis");
elmVis.style.visibility = elmVis.style.visibility == "hidden" ? "visible" : "hidden";

var elmCol = document.getElementById("toggleColor");
elmCol.style.backgroundColor = elmCol.style.backgroundColor == "blue" ? "red" : "blue";
elmCol.style.width = elmCol.style.width == "200px" ? "100px" : "200px";

var elmSel = document.getElementById("toggleFont");
elmSel.style.fontFamily = elmSel.style.fontFamily == "verdana" ? "courier" : "verdana";

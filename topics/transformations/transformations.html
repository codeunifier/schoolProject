Code Topic: CSS Transformations

Designing, Defining, and Triggering CSS3 Transformations without Custom Libraries
In this topic, I'll be going over how to transform elements using only CSS. There are a lot of attributes under the CSS "transform" property, so I'm going to outline this topic by the groupings of these attributes and cover as much as I can.
matrix
The matrix attribute defines the shape of an element by offsetting its original dimensions.
matrix 1
matrix 2
matrix 3
matrix 4
    #matrix1:hover ~ .diamond {
        transform: matrix(0.866,0.5,-0.5,0.866,0,0);
    }
    
    #matrix2:hover ~ .diamond {
        transform: matrix(0.866,0.6,-0.8,0.866,0,0);
    }
    
    #matrix3:hover ~ .diamond {
        transform: matrix(0.586,0.8,-0.8,0.586,40,40);
    }
    
    #matrix4:hover ~ .diamond {
        transform: matrix(1.25, .25, .8, .4, 0, 0);
    }
translate
The translate property allows you to specify a translational movement along the X or Y axis (or Z for 3D objects).
translate x
translate y
translate both
    #translateX:hover ~ .diamond {
        transform: translateX(300px);
    }
    
    #translateY:hover ~ .diamond {
        transform: translateY(20px);
    }
    
    #translateBoth:hover ~ .diamond {
        transform: translate(300px, 20px);
    }
scale
Similarly, an object can be scaled by its width (x), height (y), or both.
scale x
scale y
scale both
    #field31:hover ~ .diamond {
        transform: scaleX(3);
    }
    
    #field32:hover ~ .diamond {
        transform: scaleY(.3);
    }
    
    #field33:hover ~ .diamond {
        transform: scale(3, .3);
    }
rotate
You know the drill. This time, we use degrees rather than pixels. Also, I'll use an image rather than a diamond to better demonstrate. Notice that rotating on a specific axis is more like flipping the image across that axis, whereas the "rotate" attribute it more of what you might expect a rotate to be.
rotate x
rotate y
rotate both
    #rotateX:hover ~ img {
        transition-duration: 2s;
        transform: rotateX(180deg);
    }
    
    #rotateY:hover ~ img {
        transition-duration: 2s;
        transform: rotateY(180deg);
    }
    
    #rotateBoth:hover ~ img {
        transition-duration: 2s;
        transform: rotate(90deg);
    }
This combination of transitions with things like transformations is the beginning of setting up animations, which will be covered in the next topic.
